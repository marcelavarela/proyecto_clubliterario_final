# 📚 Atando Libros Sueltos — Club Literario

Sitio web desarrollado para el club de lectura "Atando Libros Sueltos", que combina 
contenido editorial (reseñas, entrevistas, agenda de encuentros) con una tienda 
de libros usados integrada.

## 🎯 Descripción

El proyecto surge como trabajo práctico del curso de JavaScript de Talento Tech, 
aplicando conceptos de programación web (DOM, eventos, async/await, APIs) a un 
caso de uso real: un sitio para una comunidad de lectores.

## ✨ Funcionalidades

- **Página de inicio**: cards con reseñas literarias, entrevistas embebidas de YouTube 
  y agenda de próximos encuentros del club.
- **Menú de navegación responsive**: con hamburguesa desplegable en dispositivos móviles.
- **Registro de usuarios**: formulario de inscripción al club.
- **Tienda de libros usados**: 
  - Catálogo de libros con filtro por categoría.
  - Carrito de compras con cálculo de totales.
  - Portadas de libros obtenidas dinámicamente desde la API de OpenLibrary 
    (por ISBN), con imágenes locales de respaldo.
  - Formulario de contacto/pedido con resumen de compra.

## 🛠️ Tecnologías utilizadas

- HTML5 semántico
- CSS3 (Flexbox, diseño responsive)
- JavaScript (ES6+): manipulación del DOM, `fetch`, Promises, `async/await`, `localStorage`
- [OpenLibrary API](https://openlibrary.org/dev/docs/api/books) para portadas de libros
- Tabler Icons para iconografía
